Clonar repositório
git clone URL

Ver status
git status

Adicionar arquivos
git add .

Fazer commit
git commit -m "mensagem"

Enviar para o GitHub
git push

Criar branch
git branch nome-da-branch

Trocar de branch
git checkout nome-da-branch

Criar e trocar branch
git checkout -b nome

Corrigir commit
git commit --amend
